# React snippet for Essential JS 2 components

![Logo](./images/synclogo.png)

This extension will insert the Essential JS 2 React components.

## React Snippets

Type the following text and press `enter` to insert the corresponding component snippet in your application.

## Editors

### AutoComplete

| Snippet       | Action       |
|---------------|--------------|
| `ej2-react-autocomplete` | Insert the snippet for Basic AutoComplete. |
| `ej2-react-autocomplete-ds-remote` | Insert the snippet for AutoComplete with `Remote DataSource`. |
| `ej2-react-autocomplete-filter` | Insert the snippet for AutoComplete with `Filtering` support. |
